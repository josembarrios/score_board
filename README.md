## Starting 🚀

### Use of the library 📋

_Import maven dependency_

```
        <dependency>
            <groupId>com.github.josembarrios</groupId>
            <artifactId>score_board_module</artifactId>
            <version>1.0-SNAPSHOT</version>
        </dependency>
```
